# Portfolio Rémy Cormon

Site web portfolio au style éditorial, minimaliste et haut de gamme, inspiré de la mode et des magazines de luxe.

## Caractéristiques

- **Design éditorial** : Esthétique noir et blanc avec tons blanc cassé
- **Typographie soignée** : Combinaison de serif (Playfair Display) et sans-serif (Inter)
- **Espaces négatifs** : Utilisation généreuse de l'espace blanc
- **Animations douces** : Transitions élégantes et fluides
- **Responsive** : Adapté à tous les écrans

## Structure

```
ePortfolio/
├── index.html          # Page d'accueil
├── about.html          # Page À propos
├── services.html       # Page Services
├── contact.html        # Page Contact
├── css/
│   └── style.css       # Styles principaux
├── js/
│   └── main.js         # Interactions JavaScript
└── images/             # Dossier pour les images
```

## Images requises

Pour un rendu optimal, ajoutez les images suivantes dans le dossier `images/` :

- `hero.jpg` : Image hero de la page d'accueil (1920x1080px minimum)
- `about.jpg` : Portrait pour la page À propos (1200x800px minimum)
- `project1.jpg` : Image projet 1 (1600x900px minimum)
- `project2.jpg` : Image projet 2 (1600x900px minimum)
- `project3.jpg` : Image projet 3 (1600x900px minimum)

**Recommandations** :
- Format : JPG ou WebP
- Qualité : Haute résolution
- Style : Noir et blanc ou tons neutres
- Esthétique : Épurée, éditoriale, luxe

## Personnalisation

### Couleurs
Les couleurs principales sont définies dans `:root` du fichier CSS :
- `--color-black` : Noir principal
- `--color-white` : Blanc principal
- `--color-off-white` : Blanc cassé
- `--color-gray` : Gris pour les textes secondaires

### Typographie
- **Serif** : Playfair Display (titres)
- **Sans-serif** : Inter (textes)

### Contenu
Modifiez le contenu directement dans les fichiers HTML :
- Nom et informations de contact
- Projets et descriptions
- Services proposés
- Textes de présentation

## Déploiement

Le site est entièrement statique et peut être déployé sur :
- GitHub Pages
- Netlify
- Vercel
- Tout hébergeur web classique

## Technologies

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (Vanilla)
- Google Fonts
